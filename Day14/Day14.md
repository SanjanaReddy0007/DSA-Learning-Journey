## 💡 Key Learnings

- Sorting is one of the most fundamental concepts in Data Structures.
- It helps arrange data in ascending or descending order.
- Different sorting algorithms perform differently depending on the input size.
- Bubble Sort repeatedly swaps adjacent elements until the array becomes sorted.
- Selection Sort selects the smallest element and places it in its correct position.
- Insertion Sort inserts each element into its correct position in the already sorted portion.
- Merge Sort follows the Divide and Conquer approach and provides O(n log n) complexity.
- Quick Sort also uses Divide and Conquer by selecting a pivot element and partitioning the array.
- Merge Sort requires extra memory, while Quick Sort sorts mostly in-place.
- Stable sorting algorithms preserve the relative order of duplicate elements.
- In-place algorithms require very little additional memory.
- Merge Sort performs consistently even for large datasets.
- Quick Sort is one of the fastest sorting algorithms in practical applications.
- Understanding time complexity helps in selecting the most efficient algorithm.
- Recursive thinking becomes stronger while implementing Merge Sort and Quick Sort.
