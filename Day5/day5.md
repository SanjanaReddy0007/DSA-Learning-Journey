### Isomorphic Strings

Definition:
Two strings are isomorphic if characters from one string can be replaced to get the other string while maintaining the order.

Example:
egg → add

Result:
Isomorphic

### Valid Anagram

Definition:
Two strings are anagrams if they contain the same characters with the same frequency.

Example:
listen → silent

Result:
Valid Anagram
